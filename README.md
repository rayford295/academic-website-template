# Yifan Yang Academic Website

This folder contains a GitHub Pages-ready personal academic website built from the
`academic-website-template` and customized using Yifan Yang's CV, photo, and public GitHub profile.

## Files

- `index.html`: the full one-page website
- `assets/photo.jpg`: profile image used in the hero section
- `assets/cv.pdf`: academic CV linked from the homepage

## Publish to GitHub Pages

1. Create or open the repository `rayford295.github.io`.
2. Upload the contents of this folder to the repository root.
3. In GitHub, open `Settings -> Pages`.
4. Set the source to the `main` branch and save.

The site will then publish at `https://rayford295.github.io/`.
